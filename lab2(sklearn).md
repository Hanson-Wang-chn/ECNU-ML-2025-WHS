# 实验二：线性回归模型训练与评估

## 实验目的：
- 理解线性回归的基本原理
- 学习Scikit-learn的使用
- 掌握模型训练、预测和评估流程

## 实验环境：
- MindSpore 2.2
- 主要库：
  - sklearn
  - numpy
  - matplotlib 

## 实验内容：
### 1. 导入Sklearn模块和辅助模块
- 导入scikit-learn中LinearRegression, mean_squared_error等模块
  
### 2. 生成数据
- 根据线性函数y = -5 * x + 0.1生成模拟数据，并在其中加入少许扰动。

  
### 3. 模型构建
- 使用Scikit-learn提供的LinearRegression()作为线性回归模型
- 采用均方差作为损失函数。
- 使用fit方法对模型进行训练优化

### 4. 模型训练
- epochs = 20，batch_size = 1

### 5. 模型预测
- 查看参数的名称、维度、数据类型、数值以及是否求导。
- 查看第一个可训练参数
- 将第一个参数数组化
- 使用训练好的模型进行预测
- 使用mean_squared_error计算均方误差
- +使用score方法计算R²分数来评估模型性能

### 6. 可视化
- 可视化模拟的样本数据、真实的线性函数和训练得到的线性模型
