# 实验四：支持向量机模型

### 实验目的：

+ 理解支持向量机的基本原理。
+ 学会使用Python中的库实现SVM模型，理解支持向量机硬分隔、软分隔和核技巧。
+ 掌握数据预处理、模型训练、评估流程和参数调优的基本步骤。

### 实验环境：

+ Python 3.x
+ 主要库：

  - Scikit-learn库
  - NumPy
  - Matplotlib

### 实验内容：

1. **随机化两类数据，运用支持向量机进行硬分隔和软分隔**
   + 使用 `datasets.make_classification` 创建一个具有两类的随机二维数据集，设定 `n_samples=200`，`n_features=2`，`n_informative=2`，`n_redundant=0`，并使用 `random_state=25` 保持生成数据的一致性。
   + 利用 `sklearn.svm.SVC` 进行 SVM 分类，尝试不同的 C 值来调节模型的分隔能力，**C值可选0.1,1,1e4或自由尝试**。C 值控制模型对误分类的容忍程度，当 C 较大时（如 `C=1e4`），SVM 更倾向于硬分隔，当 C 较小时（如 `C=0.1` 或 `C=1`），SVM 更倾向于软分隔，允许一些误分类，从而生成更平滑的决策边界。
   + 将SVC在不同C值下产生的结构可视化，绘制出数据点和决策边界
2. **基于鸢尾花数据集，运用支持向量机（SVM）进行线性分类。**
   + 从 `sklearn.datasets` 加载鸢尾花数据集，理解数据的基本结构和特征，包括样本数、特征类型和标签分布。
   + 进行数据分割，将数据集划分为训练集和测试集，并进行特征标准化处理，以便模型训练。
   + 使用 SVM 线性核对数据进行训练和预测。
   + 绘制PR曲线、ROC曲线并计算$F_1$值  
3. **基于卫星数据集，采用支持向量机（SVM）核技巧实现非线性分类。**
   + 从`sklearn.datasets`加载卫星数据集，并理解数据集的基本结构，包括特征和标签。
   + 进行数据分割，将数据集划分为训练集和测试集，并进行特征标准化处理，以便模型训练。
   + 选用适合的核函数来训练SVM模型并进行预测，理解不同核函数的影响。
   + 绘制不同核函数下的PR曲线、ROC曲线并计算$F_1$​​值  

### 数据集

+ sklearn.datasets中的`make_classification`、`sklearn.datasets.load_iris`和`sklearn.datasets.make_moons`
